# 🚀 Dev-Guide

**Power developers with AI-driven career growth.** Dev-Guide is an AI-powered career development platform that uses machine learning to provide personalized learning recommendations and mentorship matching for software developers.

## Why Dev-Guide?

* **🚀 Personalized Learning Paths**: Get tailored learning recommendations based on your skills and goals.
* **👥 Mentorship Matching**: Find the right mentor for your career growth.
* **📊 Data-Driven Insights**: Help managers allocate training budgets efficiently.
* **🔒 Scalable Architecture**: Built for high-traffic and large-scale deployments.
* **🔩 Plug-in System**: Customizable recommendation models and internal mentorship databases.
* **🚀 Cloud-Native**: Integrates with existing IDEs and CI pipelines.

## Getting Started

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python main.py
```

### Run the Tests

```bash
python -m unittest discover -s tests
```

## Deploy

### Deploy to Vercel

```bash
vercel --prod
```

# 🚀 Dev-Guide
**Power software developers with personalized learning recommendations and mentorship matching.** Dev-Guide is an AI-powered career development platform providing personalized learning recommendations and mentorship matching for software developers.

## Why Dev-Guide?
- **Personalized Learning**: Get tailored learning recommendations based on your skills and interests.
- **Mentorship Matching**: Connect with experienced mentors who can guide you in your career.
- **Data-Driven Insights**: Gain valuable insights into your learning progress and career growth.
- **Scalable Architecture**: Built to handle large volumes of users and data.
- **Integration with IDEs and CI Pipelines**: Seamlessly integrate with your existing development tools.
- **Community Support**: Join a community of like-minded developers and mentors.

## Tech Stack
* FastAPI
* Flask

## Project Structure
* `api`: API endpoints and handlers
* `business`: Business logic and models
* `docs`: Documentation and guides
* `src`: Source code and main application
* `tests`: Unit tests and integration tests

## Getting Started
```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py

# Run tests
python -m unittest discover -s tests
```

## Deploy
```bash
# Build the Docker image
docker build -t dev-guide .

# Run the Docker container
docker run -p 8000:8000 dev-guide
```
